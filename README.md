# neovim-config
My current neovim configuration 


# Installation
These instructions are for me and my setup
1. Install nvim
`winget install nvim`
2. install packer
`git clone https://github.com/wbthomason/packer.nvim "$env:LOCALAPPDATA\nvim-data\site\pack\packer\start\packer.nvim"`
3. create nvim dir in appdata local
4. clone the repo into nvim dir
5. navigate into lua/scripts/packer.lua and shout out
`so`
6. `PackerSync`
7. go back to the main dir and open init.lua and `so`
